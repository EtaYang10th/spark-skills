# Citation Authenticity Verification in BibTeX Files

## Overview

This skill covers the end-to-end process of identifying fake or hallucinated academic citations in a BibTeX file. The task pattern involves reading a `.bib` file, analyzing each entry for signs of fabrication, and producing a JSON report of fake citation titles.

Fake citations generated by LLMs share a recognizable fingerprint: they combine plausible-sounding but unverifiable metadata. The key is knowing which signals to check and how to weight them.

---

## High-Level Workflow

1. **Parse the BibTeX file** — Use `bibtexparser` to extract structured metadata (title, author, DOI, journal, year, etc.) from every entry.

2. **Clean titles** — Strip BibTeX formatting artifacts (`{}`, `\`, extra whitespace) so titles are human-readable and match expected output format.

3. **Apply heuristic filters** — Score each entry against a set of red-flag indicators (see §Heuristic Signals below). Entries that trip multiple flags are fake candidates.

4. **Verify DOIs** — For entries that have a DOI, check whether the DOI prefix belongs to a known registrant and whether the DOI resolves via `https://doi.org/`. Fake DOIs almost always use fabricated prefixes like `10.1234` or `10.5678`.

5. **Cross-reference known publications** — For entries that look suspicious but have plausible metadata, attempt a web lookup or compare against your training knowledge of landmark papers.

6. **Compile results** — Collect the cleaned titles of all fake entries, sort alphabetically, and write to the required JSON format.

7. **Validate output** — Re-read the JSON file, confirm it parses, confirm the list is sorted, and confirm the count is reasonable.

## Step 1: Parse the BibTeX File

```python
import bibtexparser
from bibtexparser.bparser import BibTexParser

def parse_bib_file(filepath: str) -> list[dict]:
    """Parse a .bib file and return a list of entry dicts."""

```

Each entry dict has keys like `"title"`, `"author"`, `"doi"`, `"journal"`, `"year"`, `"ID"` (the cite key), and `"ENTRYTYPE"`.

## Step 2: Clean BibTeX Titles

BibTeX titles often contain curly braces for case protection and LaTeX commands. Strip them for clean output.

```python
import re

def clean_title(raw_title: str) -> str:
    """Remove BibTeX formatting artifacts from a title string."""
    # Remove curly braces
    title = raw_title.replace("{", "").replace("}", "")
    # Remove common LaTeX commands
    title = re.sub(r"\\[a-zA-Z]+\s*", "", title)
    # Remove backslashes
    title = title.replace("\\", "")
    # Collapse whitespace
    title = re.sub(r"\s+", " ", title).strip()
    return title
```

## Step 3: Heuristic Signals for Fake Citations

This is the core detection logic. Each signal contributes to a suspicion score. An entry tripping 2+ strong signals is almost certainly fake.

### Signal Catalog

| Signal | Weight | Description |
|--------|--------|-------------|
| **Fake DOI prefix** | STRONG | DOI prefixes like `10.1234`, `10.5678`, `10.9999`, `10.0000` are not assigned to real registrants. Real prefixes map to known publishers (e.g., `10.1038` = Nature, `10.1126` = Science, `10.1145` = ACM). |
| **Missing DOI** | WEAK | Many real papers lack DOIs (older papers, preprints, workshops), but combined with other signals it's informative. |
| **Generic author names** | MODERATE | Names like "John Smith", "Alice Johnson", "Emily Wilson", "Robert Taylor" — common English placeholder names that LLMs default to. Real papers have distinctive author combinations. |
| **Unverifiable journal/venue** | STRONG | Journal names that sound plausible but don't exist: "Journal of Computational Linguistics" (the real one is "Computational Linguistics"), "AI Research Journal", "International Journal of Advanced Computing". |
| **Vague or overly broad title** | MODERATE | Titles like "Advances in X for Y" or "A Comprehensive Review of Z" without specificity. Real papers tend to have more precise titles. |
| **Round/suspicious page numbers** | WEAK | Page ranges like `112-130` or `200-220` that look fabricated. Not reliable alone. |
| **DOI does not resolve** | STRONG | If a DOI returns 404 from `https://doi.org/{doi}`, it's fake. But rate-limit your requests. |

### Known Fake DOI Prefixes

```python
KNOWN_FAKE_DOI_PREFIXES = {
    "10.1234",
    "10.5678",
    "10.9999",
    "10.0000",
    "10.1111/fake",
    "10.2222",
    "10.3333",
    "10.4444",
    "10.6666",
    "10.7777",
    "10.8888",
}

def has_fake_doi_prefix(doi: str) -> bool:
    """Check if a DOI uses a known fabricated prefix."""
    if not doi:
        return False
    for prefix in KNOWN_FAKE_DOI_PREFIXES:
        if doi.startswith(prefix):
            return True
    # Also flag any DOI whose prefix is a simple 4-digit round number
    # Real DOI prefixes are assigned by registration agencies and are not round
    parts = doi.split("/")
    if len(parts) >= 2:
        prefix = parts[0]  # e.g. "10.1234"
        registrant = prefix.replace("10.", "")
        if registrant.isdigit() and len(registrant) == 4:
            last_three = registrant[1:]
            if last_three == "000" or registrant == registrant[0] * 4:
                return True
    return False
```

### Generic Author Detection

```python
GENERIC_FIRST_NAMES = {
    "john", "jane", "alice", "bob", "emily", "robert", "michael",
    "sarah", "david", "james", "mary", "carlos", "aisha", "wei",
    "li", "anna", "peter", "mark", "laura", "chris", "daniel",
}

GENERIC_LAST_NAMES = {
    "smith", "johnson", "williams", "brown", "jones", "davis",
    "miller", "wilson", "moore", "taylor", "anderson", "thomas",
    "jackson", "white", "harris", "martin", "thompson", "garcia",
    "martinez", "robinson", "clark", "rodriguez", "lewis", "lee",
    "walker", "hall", "allen", "young", "king", "wright", "lopez",
    "hill", "scott", "green", "adams", "baker", "gonzalez", "nelson",
    "carter", "mitchell", "perez", "roberts", "turner", "phillips",
    "campbell", "parker", "evans", "edwards", "collins", "stewart",
    "ramirez", "patel",
}

def author_genericity_score(author_str: str) -> float:
    """
    Return a score 0-1 indicating how 'generic' the author names are.
    Higher = more suspicious.
    """
    # bibtexparser joins authors with " and "
    authors = [a.strip() for a in author_str.lower().replace("\n", " ").split(" and ")]
    if not authors:
        return 0.0

    generic_count = 0
    for author in authors:
        parts = re.split(r"[,\s]+", author)
        parts = [p for p in parts if p]
        first_match = any(p in GENERIC_FIRST_NAMES for p in parts)
        last_match = any(p in GENERIC_LAST_NAMES for p in parts)
        if first_match and last_match:
            generic_count += 1

    return generic_count / len(authors)
```

### Fake Journal Detection

```python
KNOWN_FAKE_JOURNALS = {
    "journal of computational linguistics",  # Real one is just "Computational Linguistics"
    "ai research journal",
    "international journal of advanced computing",
    "journal of artificial intelligence research and applications",
    "advanced computing review",
    "journal of machine learning applications",
    "international journal of deep learning",
    "neural computing and applications review",
    "journal of blockchain technology",
    "journal of data science and analytics",
}

KNOWN_REAL_JOURNALS_PARTIAL = {
    "nature", "science", "cell", "lancet", "pnas",
    "proceedings of the", "advances in neural information",
    "international conference on", "annual meeting of the",
    "transactions on", "journal of the acm", "communications of the acm",
    "ieee", "acm", "springer", "elsevier", "arxiv",
    "computational linguistics",  # the real MIT Press journal
    "findings of", "conference on empirical methods",
}

def is_suspicious_journal(journal: str) -> bool:
    """Check if a journal name looks fabricated."""
    if not journal:
        return False  # Missing journal is handled separately
    j_lower = journal.lower().strip()

    # Direct match against known fakes
    if j_lower in KNOWN_FAKE_JOURNALS:
        return True

    # Check if it matches any known real journal pattern
    for pattern in KNOWN_REAL_JOURNALS_PARTIAL:
        if pattern in j_lower:
            return False

    # Heuristic: very generic-sounding journal names
    # "Journal of X" where X is a broad field with no specificity
    generic_patterns = [
        r"^journal of \w+ \w+$",  # "Journal of Two Words"
        r"^international journal of \w+$",
        r"^\w+ research journal$",
        r"^\w+ review$",
    ]
    for pat in generic_patterns:
        if re.match(pat, j_lower):
            return True

    return False
```

## Step 4: DOI Resolution Verification

Use HTTP HEAD requests to `doi.org` to check if a DOI actually resolves. This is the strongest signal but requires network access.

```python
import requests

def doi_resolves(doi: str, timeout: int = 10) -> bool:
    """
    Check if a DOI resolves to a real resource.
    Returns True if it resolves, False if 404/error.
    Returns None if network is unavailable (don't penalize).
    """
    if not doi:
        return None

    url = f"https://doi.org/{doi}"
    try:
        resp = requests.head(url, allow_redirects=True, timeout=timeout)
        return resp.status_code == 200
    except requests.RequestException:
        # Network issue — don't use this signal
        return None
```

**Important**: Rate-limit DOI checks. If you have 20+ entries, add a small delay between requests or only check entries that already have 1+ heuristic flags.

## Step 5: Composite Scoring and Decision

```python
def evaluate_entry(entry: dict) -> dict:
    """
    Evaluate a single BibTeX entry for authenticity.
    Returns a dict with the entry ID, cleaned title, score, and flags.
    """
    flags = []
    score = 0.0

    title = clean_title(entry.get("title", ""))
    doi = entry.get("doi", "").strip()
    author = entry.get("author", "")
    journal = entry.get("journal", entry.get("booktitle", ""))

    # DOI prefix check (STRONG)
    if doi and has_fake_doi_prefix(doi):
        flags.append("fake_doi_prefix")
        score += 3.0
    elif not doi:
        flags.append("missing_doi")
        score += 0.5

    # Author genericity (MODERATE)
    if author:
        gen_score = author_genericity_score(author)
        if gen_score >= 0.8:
            flags.append("generic_authors")
            score += 2.0
        elif gen_score >= 0.5:
            flags.append("somewhat_generic_authors")
            score += 1.0

    # Journal check (STRONG)
    if journal and is_suspicious_journal(journal):
        flags.append("suspicious_journal")
        score += 3.0

    # DOI resolution (STRONG, optional)
    if doi and "fake_doi_prefix" not in flags:
        resolves = doi_resolves(doi)
        if resolves is False:
            flags.append("doi_does_not_resolve")
            score += 3.0
        elif resolves is True:
            # Positive signal — reduce suspicion
            score -= 1.0

    return {
        "id": entry.get("ID", "unknown"),
        "title": title,
        "score": score,
        "flags": flags,
    }
```

**Decision threshold**: An entry with `score >= 4.0` is almost certainly fake. In practice, fake LLM-generated citations typically score 5.0–8.0 because they trip multiple strong signals simultaneously (fake DOI prefix + generic authors + fake journal).

Real papers rarely score above 1.0. A score of 2.0–3.0 is ambiguous and warrants manual review or web search.

## Step 6: Write the Output

```python
import json

def write_answer(fake_titles: list[str], output_path: str):
    """Write the answer JSON with alphabetically sorted fake titles."""
    fake_titles_sorted = sorted(fake_titles)
    result = {"fake_citations": fake_titles_sorted}

    with open(output_path, "w", encoding="utf-8") as f:
        json.dump(result, f, indent=2, ensure_ascii=False)

    # Verification: re-read and validate
    with open(output_path, "r", encoding="utf-8") as f:
        check = json.load(f)
    assert "fake_citations" in check
    assert isinstance(check["fake_citations"], list)
    assert check["fake_citations"] == sorted(check["fake_citations"]), "Titles must be sorted"
    print(f"Wrote {len(check['fake_citations'])} fake citations to {output_path}")
    print(f"Sorted: {check['fake_citations'] == sorted(check['fake_citations'])}")
```

## Step 7: Validation Checklist

Before finalizing, verify:

- [ ] Output file is valid JSON (parseable by `json.load`)
- [ ] Top-level key is `"fake_citations"` (exact spelling)
- [ ] Value is a list of strings (not dicts, not nested)
- [ ] Titles are cleaned (no `{}`, no `\`, no extra whitespace)
- [ ] Titles are sorted alphabetically (Python default `sorted()`)
- [ ] Count is reasonable (typically 2–5 fakes in a file of 10–20 entries)
- [ ] No real papers are included (double-check any borderline cases)

## Common Pitfalls

### 1. Confusing real journals with similar-sounding fake ones
**"Computational Linguistics"** is a real MIT Press journal. **"Journal of Computational Linguistics"** is not. Always check the exact name, not just keywords.

### 2. Flagging old papers for missing DOIs
Papers from before ~2000 often lack DOIs. Missing DOI alone is a weak signal — never classify an entry as fake based solely on missing DOI.

### 3. Not cleaning BibTeX title formatting
Validators compare exact strings. If you leave `{Deep Learning}` instead of `Deep Learning`, the match fails. Always strip `{}` and `\` from titles.

### 4. Forgetting to sort alphabetically
The expected output requires alphabetical sorting. Use Python's `sorted()` on the final list.

### 5. Over-relying on DOI resolution without heuristics
Network requests can fail, be rate-limited, or time out. Always run heuristic checks first. DOI resolution is a confirmation step, not the primary detector.

### 6. Treating well-known papers as suspicious
Landmark papers (AlphaFold, CRISPR, Watson & Crick, etc.) are real even if their metadata looks unusual. If you recognize a paper from your training data as a widely-cited real publication, trust that knowledge.

### 7. Including the BibTeX cite key instead of the title
The output must contain cleaned **titles**, not cite keys like `smith2020ai`.

### 8. Not handling `booktitle` for conference papers
Conference papers use `booktitle` instead of `journal`. Check both fields when evaluating venue authenticity.

```python
    """Remove BibTeX formatting artifacts from a title string."""
    title = raw_title.replace("{", "").replace("}", "")
    title = re.sub(r"\\[a-zA-Z]+\s*", "", title)
    title = title.replace("\\", "")
    title = re.sub(r"\s+", " ", title).strip()
    return title


def parse_bib_file(filepath: str) -> list:
    """Parse a .bib file and return a list of entry dicts."""
    parser = BibTexParser(common_strings=True)
    parser.customization = convert_to_unicode
    with open(filepath, "r", encoding="utf-8") as f:

        return False

    if j_lower in KNOWN_FAKE_JOURNALS:
        return True

        r"^international journal of \w+$",
        r"^\w+ research journal$",

def doi_resolves(doi: str, timeout: int = 10) -> bool:
    """
    Check if a DOI resolves. Returns True/False/None.
    None means network unavailable (don't use as signal).
    """
    if not HAS_REQUESTS or not doi:
        return None
    url = f"https://doi.org/{doi.strip()}"
    try:
        resp = requests.head(url, allow_redirects=True, timeout=timeout)
        return resp.status_code == 200
    except requests.RequestException:
        return None


# ── Core Evaluation ────────────────────────────────────────────────────────

def evaluate_entry(entry: dict, check_doi_resolution: bool = False) -> dict:
    """
    Score a BibTeX entry for authenticity.
    Returns dict with id, title, score, flags, and is_fake.
    """
    flags = []
    score = 0.0

    title = clean_title(entry.get("title", ""))
    doi = entry.get("doi", "").strip()
    author = entry.get("author", "")
    journal = entry.get("journal", entry.get("booktitle", ""))

    # ── DOI prefix check (STRONG signal) ──
    if doi and has_fake_doi_prefix(doi):
        flags.append("fake_doi_prefix")
        score += 3.0
    elif not doi:
        flags.append("missing_doi")
        score += 0.5

    # ── Author genericity (MODERATE signal) ──
    if author:
        gen = author_genericity_score(author)
        if gen >= 0.8:
            flags.append(f"generic_authors (score={gen:.2f})")
            score += 2.0
        elif gen >= 0.5:
            flags.append(f"somewhat_generic_authors (score={gen:.2f})")
            score += 1.0

    # ── Journal/venue check (STRONG signal) ──
    if journal and is_suspicious_journal(journal):
        flags.append("suspicious_journal")
        score += 3.0
    elif not journal:
        flags.append("missing_journal")
        score += 0.5

    # ── DOI resolution (STRONG confirmation, optional) ──
    if check_doi_resolution and doi and "fake_doi_prefix" not in flags:
        time.sleep(0.5)  # Rate limiting
        resolves = doi_resolves(doi)
        if resolves is False:
            flags.append("doi_does_not_resolve")
            score += 3.0
        elif resolves is True:
            score -= 1.0  # Positive confirmation

    return {
        "id": entry.get("ID", "unknown"),
        "title": title,
        "score": score,
        "flags": flags,
        "is_fake": score >= FAKE_SCORE_THRESHOLD,
    }


# ── Main Pipeline ──────────────────────────────────────────────────────────

def find_fake_citations(
    bib_path: str,
    output_path: str,
    check_doi_resolution: bool = False,
    verbose: bool = True,
) -> list:
    """
    Full pipeline: parse → evaluate → write results.
    Returns the list of fake titles.
    """
    # Step 1: Parse
    entries = parse_bib_file(bib_path)
    if verbose:
        print(f"Parsed {len(entries)} entries from {bib_path}")

    # Step 2: Evaluate each entry
    results = []
    for entry in entries:
        result = evaluate_entry(entry, check_doi_resolution=check_doi_resolution)
        results.append(result)
        if verbose:
            status = "FAKE" if result["is_fake"] else "ok"
            print(f"  [{status:4s}] {result['id']:30s}  score={result['score']:.1f}  flags={result['flags']}")

    # Step 3: Collect fake titles
    fake_titles = [r["title"] for r in results if r["is_fake"]]
    fake_titles_sorted = sorted(fake_titles)

    if verbose:
        print(f"\nIdentified {len(fake_titles_sorted)} fake citation(s):")
        for t in fake_titles_sorted:
            print(f"  - {t}")

    # Step 4: Write output
    output = {"fake_citations": fake_titles_sorted}
    with open(output_path, "w", encoding="utf-8") as f:
        json.dump(output, f, indent=2, ensure_ascii=False)

    # Step 5: Validate output
    with open(output_path, "r", encoding="utf-8") as f:
        check = json.load(f)
    assert "fake_citations" in check, "Missing 'fake_citations' key"
    assert isinstance(check["fake_citations"], list), "'fake_citations' must be a list"
    assert check["fake_citations"] == sorted(check["fake_citations"]), "Titles must be sorted"

    if verbose:
        print(f"\nWrote valid JSON to {output_path}")
        print(f"  Count: {len(check['fake_citations'])}")
        print(f"  Sorted: True")

    return fake_titles_sorted

        bib_path=args.bib,
        output_path=args.out,
        check_doi_resolution=args.check_doi,
        verbose=not args.quiet,
    )

    sys.exit(0 if fake else 0)  # Always exit 0; the JSON file is the deliverable
```

## Decision Tree Summary

```
For each BibTeX entry:
│
├─ DOI present?
│  ├─ YES → DOI prefix in KNOWN_FAKE_DOI_PREFIXES?
│  │         ├─ YES → +3.0 (strong fake signal)
│  │         └─ NO  → optionally verify resolution
│  └─ NO  → +0.5 (weak signal, common for old/preprint papers)
│
├─ Authors generic? (both first AND last name in generic lists)
│  ├─ ≥80% of authors generic → +2.0
│  ├─ ≥50% of authors generic → +1.0
│  └─ <50% → +0.0
│
├─ Journal/venue suspicious?
│  ├─ In KNOWN_FAKE_JOURNALS → +3.0
│  ├─ Matches generic pattern and NOT in known-real fragments → +3.0
│  └─ Recognized real venue → +0.0
│
└─ Total score ≥ 4.0? → FAKE
   Total score < 4.0? → REAL
```

Typical fake entry scores: 5.0–8.0 (fake DOI + generic authors + fake journal).
Typical real entry scores: 0.0–1.0 (valid DOI, known authors, recognized venue).
